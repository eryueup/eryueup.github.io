Syncing

Syncing (源码) 是一个 VSCode 扩展，它能在多台设备之间同步您的所有 VSCode 配置（借助了 GitHub Gist）。

具体详情查看官方描述。下面只记录自己的使用步骤：Windows->Windows

安装Syncing扩展

本地和需要更换到的目标设备上vscode都需要安装

快速使用

申请一个有Gist权限的GitHub Personal Access Token，记住此令牌

在 VSCode 的命令面板中，您可以输入 upload、download（或者 syncing）来快速搜索和执行 Syncing 的所有命令。

使用upload根据提示输入Token、默认ID即可，上传完成

使用download根据提示输入，等待下载配置

上传简要说明

用户设置 快捷键扩展 语言设置以及所有代码片段（Snippets）

自动创建Gist保存vscode的配置

也可以自定义上传配置项和扩展

跨设备需另行注意

下载简要说明

下载配置时会覆盖您的本地配置（以云端为准，精确同步）

自动安装、升级和删除扩展
